# RFIC布局数据生成器GUI应用 - 完整使用指南

## 📋 项目概述

这是一个基于Python Tkinter的图形化RFIC布局数据生成工具，专为Keysight ADS 2025 Python API设计。支持创建、编辑、可视化和批量处理RFIC布局数据，用于电磁仿真和电路设计。

## 🎯 核心功能

### 1. 布局设计
- **多层布局创建**：支持多图层设计
- **随机布局生成**：基于概率的随机填充
- **端口定义**：可视化端口位置和连接
- **实时预览**：交互式布局预览

### 2. 数据管理
- **批量操作**：支持批量生成、导入、导出
- **数据增强**：90°/180°/270°旋转，水平/垂直翻转
- **跨平台文件操作**：Windows/Linux兼容
- **JSON格式存储**：标准化数据交换

### 3. 可视化
- **图层控制**：独立显示/隐藏各层
- **端口标记**：可视化端口位置和编号
- **颜色编码**：不同图层使用不同颜色
- **网格显示**：16x16像素网格对齐

## 🖥️ 界面布局

### 左侧面板 - 控制与信息
- **元数据设置**：工艺信息、像素尺寸、描述
- **图层管理**：添加/删除/重排序图层
- **端口定义**：端口ID、名称、图层、边缘位置

### 中间面板 - 可视化预览
- **实时预览**：当前布局的图形化显示
- **图层过滤**：通过复选框控制图层可见性
- **端口显示**：显示端口位置和编号
- **图例**：颜色编码的图层说明

### 右侧面板 - 文件管理
- **设计列表**：所有保存的设计
- **批量操作**：
  - 批量生成随机布局
  - 批量导出选中设计
  - 批量导入JSON文件
- **数据增强**：旋转和翻转变换

## 📁 数据文件格式

### JSON文件结构

```json
{
  "design_id": "design_20250725_151829_200925_0",
  "metadata": {
    "process": "GF 22FDX+",
    "pixel_size_um": 14.0,
    "description": "Generated by LayoutGeneratorApp",
    "base_matrix_shape": [16, 16],
    "layers_used": ["metal1", "metal2", "via1"]
  },
  "layout_matrices": {
    "metal1": [[0, 1, 0, ...], [1, 1, 0, ...], ...],
    "metal2": [[1, 0, 1, ...], [0, 1, 1, ...], ...],
    "via1": [[0, 0, 1, ...], [0, 1, 0, ...], ...]
  },
  "port_definitions": [
    {
      "port_id": 1,
      "name": "P1",
      "layer": "metal1",
      "edge": "left",
      "position_index": 8
    },
    {
      "port_id": 2,
      "name": "P2",
      "layer": "metal2",
      "edge": "right",
      "position_index": 8
    }
  ]
}
```

### 字段说明

#### metadata 字段
- **process**: 工艺节点 (如 "GF 22FDX+")
- **pixel_size_um**: 像素尺寸 (微米)
- **description**: 设计描述
- **base_matrix_shape**: 基础矩阵尺寸 [行, 列]
- **layers_used**: 使用的图层列表

#### layout_matrices 字段
- **键**: 图层名称
- **值**: 16x16的numpy数组，1表示有金属，0表示空白

#### port_definitions 字段
- **port_id**: 端口唯一标识符 (整数)
- **name**: 端口名称 (如 "P1", "P2")
- **layer**: 端口所在图层
- **edge**: 端口位置 ("left", "right", "top", "bottom")
- **position_index**: 端口在边缘上的位置索引 (0-15)

## 🚀 使用教程

### 快速入门

1. **启动应用**
   ```bash
   python RFIC布局数据生成器GUI应用.py
   ```

2. **创建新设计**
   - 添加图层：在左侧"层管理"输入图层名，点击"添加"
   - 生成布局：点击"生成随机版图"
   - 添加端口：设置端口参数，点击"添加"

3. **保存设计**
   - 点击"保存到列表"将当前设计保存
   - 点击"保存JSON到磁盘"导出单个设计

### 高级操作

#### 批量生成
1. 设置好模板设计
2. 在右侧"批量生成"输入数量
3. 点击"批量生成"创建多个随机设计

#### 批量导入
1. 点击"批量导入JSON"
2. 选择多个JSON文件
3. 自动导入并处理重复ID

#### 数据增强
1. 在列表中选择设计（可多选）
2. 使用旋转按钮：90°/180°/270°
3. 使用翻转按钮：水平/垂直
4. 新设计自动添加到列表

#### 图层过滤
1. 在预览区域上方查看图层复选框
2. 勾选/取消勾选控制图层可见性
3. "全部层"复选框控制所有图层

- **Python**: 3.7+
- **依赖库**: 
  - tkinter (内置)
  - numpy >= 1.19.0
  - matplotlib >= 3.3.0

- **操作系统**: Windows, Linux, macOS
- **编码**: UTF-8

### 性能参数
- **最大设计数量**: 无限制（内存允许）
- **矩阵大小**: 固定16x16像素
- **图层数量**: 无限制
- **端口数量**: 无限制

## 🚨 常见问题

### 导入失败
- **检查文件格式**: 确保是有效的JSON
- **验证数据结构**: 确认包含必要字段
- **处理编码问题**: 使用UTF-8编码

### 显示异常
- **重置视图**: 重新加载设计
- **检查图层**: 确认图层可见性设置
- **更新matplotlib**: 确保使用最新版本

### 内存问题
- **分批处理**: 大量数据时分批导入
- **清理缓存**: 定期清空不再使用的设计
- **优化数据结构**: 避免不必要的冗余
