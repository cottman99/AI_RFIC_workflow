# Legacy Variants

This directory contains archived GUI variants that are no longer part of the primary public-facing path.

## Included Files

- `layout_generator_gui.py`: baseline GUI variant
- `layout_generator_gui_enhanced.py`: enhanced GUI variant
- `test_enhanced_features.py`: historical smoke test for the enhanced variant

## Repository Position

These files are retained for:

- implementation comparison
- maintenance recovery
- feature backtracking

They are not the recommended place for a new user to start.

For normal usage, return to:

- `../layout_generator_main.py`
- `../layout_generator_gui_super_enhanced.py`
