# RFIC模型验证工具 - GUI版本

## 简介

这是一个基于tkinter的图形化界面工具，用于验证RFIC电磁仿真模型的预测结果。相比命令行版本，GUI版本提供了更直观的操作界面，支持文件路径的手动选择和参数配置。

## 功能特点

- **图形化界面**：直观的文件选择和参数配置
- **文件选择器**：支持HDF5数据集和模型文件的浏览选择
- **参数配置**：
  - 样本索引输入
  - 绘图模式选择（全图模式/子图模式）
  - 输出图片路径选择（可选）
- **实时状态显示**：验证过程的详细日志输出
- **错误处理**：友好的错误提示和异常处理

## 使用方法

### 1. 启动GUI工具

```bash
# 在命令行中运行
cd f:\pfli\Code\PY_code\ADS_Python_API\AI_RFIC_workflow\Pytorch_Model\src\tools
python verify_model_gui.py
```

### 2. 界面操作步骤

1. **选择HDF5数据集文件**：
   - 点击"浏览..."按钮选择HDF5格式的数据集文件
   - 支持文件扩展名：`.h5`, `.hdf5`

2. **选择模型文件**：
   - 点击"浏览..."按钮选择训练好的PyTorch模型文件
   - 支持文件扩展名：`.pth`, `.pt`

3. **设置样本索引**：
   - 在文本框中输入要验证的数据样本索引（默认为0）

4. **选择绘图模式**：
   - **全图模式**：所有S参数绘制在一张图上，便于对比
   - **子图模式**：每个S参数使用独立的子图，便于详细分析

5. **设置输出图片（可选）**：
   - 点击"浏览..."按钮选择保存对比图的路径
   - 支持格式：`.png`, `.jpg`
   - 留空则不保存图片，仅显示

6. **开始验证**：
   - 点击"开始验证"按钮执行验证过程

### 3. 验证过程

验证过程将在状态显示区域实时输出日志信息：

- 设备配置信息
- 数据集加载状态
- 模型加载状态
- 样本获取信息
- 模型推理进度
- 验证损失计算结果
- 图形绘制状态

### 4. 结果查看

- **图形显示**：验证完成后自动弹出S参数对比图
- **图片保存**：如果设置了输出路径，图片将自动保存
- **详细日志**：所有操作步骤和结果都会显示在状态文本框中

## 错误处理

GUI版本提供了完善的错误处理：

- **文件未找到**：提示用户文件路径错误
- **索引超出范围**：提示样本索引无效
- **模型加载失败**：显示具体的错误信息
- **绘图错误**：捕获并显示绘图过程中的异常

## 技术要求

- Python 3.6+
- PyTorch
- tkinter（Python标准库）
- matplotlib
- pandas

## 注意事项

1. 确保所有依赖库已正确安装
2. 模型文件必须与数据集格式匹配
3. 样本索引应在数据集范围内（0到len(dataset)-1）
4. 输出图片路径需要写入权限
5. 验证过程可能需要一些时间，请耐心等待

## 故障排除

### 常见问题

1. **"ModuleNotFoundError"**：
   - 确保已安装所有依赖库
   - 运行：`pip install torch matplotlib pandas`

2. **"FileNotFoundError"**：
   - 检查文件路径是否正确
   - 确保文件存在且有读取权限

3. **"IndexError"**：
   - 检查样本索引是否在有效范围内
   - 使用较小的索引值（如0, 1, 2等）

4. **绘图窗口不显示**：
   - 确保matplotlib后端配置正确
   - 尝试在命令行运行以查看错误信息
